What is Multi-device Timing?

Timing

  • ensure time-sensitive stuff happens at the correct time
  • interactive control for temporal aspects

Scopes

  • intra-page timing and control : within browser context
  • cross-page timing and control : across browser contexts

Note : Scope of multi-device timing includes both scopes!

Timing Challenges

Challenge Use-case
across media sources multi-angle video or ad-insertion
